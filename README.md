## 🚀 Getting Setup

To run the website locally:

```bash
git clone https://github.com/jaydoncarter/jaydoncarter.github.io.git
cd jaydoncarter.github.io

# Windows (Git Bash)
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```
